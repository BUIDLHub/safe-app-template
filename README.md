# Gnosis Safe App React Template

## Getting started

- Create a new React app using this template
  - `npx create-react-app my-safe-app --template @gnosis.pm/cra-template-safe-app`
- Open your new app
  - `cd my-safe-app`
- Follow the [README](template/README.md) of your Safe app
